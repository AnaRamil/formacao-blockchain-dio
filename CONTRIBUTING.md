# Contribuindo

Obrigado pelo interesse em contribuir para o projeto Wallet Manager! Abaixo estão algumas orientações simples para facilitar colaborações.

## Como abrir uma issue

- Verifique se já existe uma issue igual antes de abrir uma nova.
- Forneça um título claro e passos para reproduzir, junto com o ambiente (OS, Node.js versão).

## Como enviar um Pull Request

1. Fork o repositório e crie uma branch com um nome descritivo, por exemplo `fix/readme-typo` ou `feature/add-cli`.
2. Faça commits pequenos e com mensagens claras.
3. Abra o PR descrevendo o que foi alterado e por quê. Referencie issues relacionadas.

## Estilo de código

- Siga o estilo já presente no projeto (JavaScript padrão). Mantenha código legível e conciso.
- Execute `npm test` (se houver) antes de submeter.

## Segurança

- Não inclua chaves privadas, mnemonics, senhas ou outros segredos em commits.

## Revisão

- PRs podem ser revisados e pedir mudanças; faça alterações e atualize o mesmo PR.

## Agradecimentos

Contribuições são bem-vindas — obrigado por ajudar a melhorar este projeto!
